# Сайт кафе «Причал»

Одностраничный сайт с меню, корзиной и оформлением заказа в WhatsApp/Telegram.
Не требует сервера и базы данных — это статический сайт (HTML/CSS/JS).

## Структура

```
site/
├── index.html        ← сама страница
├── css/styles.css    ← оформление
├── js/
│   ├── config.js     ← КОНТАКТЫ И НАСТРОЙКИ (правьте здесь)
│   ├── menu-data.js  ← МЕНЮ: блюда, цены, фото
│   └── app.js        ← логика (меню, корзина, заказ)
└── assets/img/       ← логотип и фотографии блюд
```

## Что нужно заполнить

Откройте `js/config.js` и впишите свои данные:
- город, адрес, часы работы, телефон;
- **WhatsApp** (номер цифрами, напр. `79280000000`) и/или **Telegram** (username без @) —
  именно сюда будут приходить заказы;
- по желанию: ссылки на соцсети, карту, условия доставки.

После заполнения кнопка «Оформить заказ» в корзине сама сформирует сообщение со
списком блюд и суммой и откроет ваш мессенджер.

## Как изменить меню

Всё меню — в `js/menu-data.js`. Каждое блюдо:
```js
{ name: 'Название', price: 390 }                       // одна цена
{ name: 'Название', desc: 'описание', img: 'foto.jpg' } // с описанием и фото
{ name: 'Капучино', variants: [                         // несколько объёмов
    { label: '200 мл', price: 150 },
    { label: '300 мл', price: 170 } ] }
```
Фото кладите в `assets/img/` и указывайте имя файла в поле `img`.

## Как посмотреть локально

Запустите в папке `site` любой статический сервер, например:
```
python -m http.server 5500
```
и откройте http://localhost:5500

## Как опубликовать (бесплатно)

- **Netlify Drop** — netlify.com/drop, просто перетащите папку `site`.
- **GitHub Pages** — залейте папку в репозиторий и включите Pages.
- К любому из вариантов потом можно привязать свой домен.
